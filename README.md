# CMP5361-Computer-Mathematics
# CMP5361 Task 1 – Hex, Memory, and Stack Calculator (Option 1)

## Python version
Python 3.11+

## How to run the programme
From the folder containing `programA.py`:

```bash
python programA.py
````
This runs ProgramA.py

When the menu appears, choose:

* `1` to run Option 1: Convert (decimal → hex and 16-bit binary + SIGNED16)
* `2` to run Option 2: Little-endian pack/unpack (16-bit)
* `3` to run Option 3: ASCII memory dump
* `0` to exit

## How to run the tests (unittest)

From the same folder:

```bash
python -m unittest -v
```

This runs `testA.py`, which imports and tests the Option 1 functions from `programA.py`.
